# alrg-app-legal

Public-facing legal pages for the **Gia** application, an internal-use
software application operated by **ALRG QA Inc**, doing business as
**Ariella Lukasz Realty Group**.

- [End-User License Agreement](https://lukasz0221-giaos.github.io/alrg-app-legal/eula.html)
- [Privacy Policy](https://lukasz0221-giaos.github.io/alrg-app-legal/privacy.html)

These URLs are referenced from third-party app registrations
(e.g., Intuit Developer Portal for QuickBooks Online API access).

## Why this repo exists

The main GiaOS repository is private. Intuit and other API providers
require publicly-accessible EULA and Privacy Policy URLs before
granting production API credentials. This separate public repo
hosts those two documents and nothing else.
